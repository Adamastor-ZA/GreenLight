Option 1 — Auto-fetch (best for daily use)

Ask your Clubmaster administrator for your club's teesheet.co.za booking URL. Paste it into the Auto-fetch tab. Click Fetch each morning or when the tee sheet is updated. GreenLight will pull all bookings automatically.

Option 2 — CSV paste (most reliable fallback)

In Clubmaster desktop software: Reports → Tee Sheet → Export to CSV. Open the file, select all, copy, and paste into the Paste CSV export tab. Works even if your club's portal has restricted web access.

Option 3 — Manual entry

Use the Add booking form below for ad-hoc entries or small estates with few daily bookings.

Need API access?

Contact Clubmaster at clubmaster.africa to ask about CSV export configuration or API integration options for your subscription.
